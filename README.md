# TodoList

TodoList app written using React, SASS and TypeScript

## Features
- Adding tasks
- Task sorting
- Deleting tasks

## How to use it?
1. Clone the repository and go to the project folder
2. Open the project in terminal
```
cd "dirname"
```
3. Start the project using __npm start__
